# Student Management System
A Python OOP project that manages students, scores and classrooms.

## Features
- Add, remove and search students
- Add scores by subject
- Calculate average scores
- Find top student
- Save data to file

## Concepts used
- OOP and inheritance
- File handling
- Error handling
- Dictionary and list comprehension# student_management_system
